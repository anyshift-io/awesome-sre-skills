# Contributing to awesome-sre-skills

This list is open to community contributions.

## How to add an entry

1. Fork the repo.
2. Add your entry to the right section in `README.md`, alphabetically within the section. Format:

   ```
   - [Name](https://example.com) - One-sentence description ending with a period.
   ```

3. Open a PR with a title like `Add <name> to <section>`.

## What we accept

- Skill libraries, MCP servers, and tools specifically aimed at SRE, reliability, or incident work for AI agents.
- Reading material that meaningfully shaped how the AI-SRE category thinks.
- Other curated lists adjacent to ours.

## What we don't accept

- Self-promotional links to commercial products with no open component.
- Dead repos (last commit more than 18 months ago with no explanation).
- Tools without an obvious connection to SRE or reliability.

## Maintenance constraint

Anyshift maintains this list. If quarterly maintenance slips, the list gets archived rather than left to rot. This is a deliberate trade-off: a small list that stays current beats a large list that decays.
